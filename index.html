    Про курс
    Цю навчальну програму ми створюємо разом із Копенгагенським
    інститутом нейрокреативності (CINC).
    Креативність лежить в основі людської поведінки та саме вона є ключем
    для розв’язання проблем та створення інновацій.
    Протягом останнього десятиліття у CINC досліджували, як працює наш мозок, коли
    ми креативимо. Тепер на основі цих досліджень інститут допомагає підсилити свою
    креативність всімохочим: підприємцям, бізнесу, політикам, урядовцям, членам
    громадського суспільства, студентам, вченим. Інститут співпрацює з:
    Стенфордська школа інженерії
    Кембриджський університет
    медіа TEDxCopenhagen
    асоціація університетів Venture Cup Denmark
    Збройні сили Норвегії
    Копенгагенський інститут дизайну взаємодії
    Розвиток підприємництва є ключовим для відновлення економіки наших громад,
    міст і країни.     Наша програма - можливість для вас відкрити власний потенціал
    креативного мислення та навчитися успішно реалізовувати ідеї в бізнесі,
    фокусуватися на креативних аспектах кожного етапу підприємницького шляху.
    КРЕАТИВНЕ
    ПІДПРИЄМНИЦТВО ЦЕ:
    • 8 модулів навчання за ціною одного завдяки партнерській підтримці;
    • Кейсовий візит 20 учасників до європейських партнерів для обміну підприємницьким досвідом*;
    • Сертифікат “ANC Practitioner” від Копенгагенського Інституту Нейрокреативності;
    • Українські викладачі з досвідом у глобальних міжнародних компаніях та розумінням локального контексту;
    • Можливість отримати консультації від експертів щодо ваших ідей та обмінюватись досвідом з колегами підприємцями.
    * через обмеження воєнного стану тур лише для жінок підприємиць
    Використовуйте свою креативність для впровадження інноваційних підходів
    у власній справі, створення продуктів з доданою вартістю та успішної
    конкуренції на ринку!
    Для кого ця
    навчальна програма
    • Для підприємців, які  “вперлися в стелю” в розвитку власної справи;
    • Для підприємців, які мають запит на перехід на наступний етап розвитку;
    • Для менеджерів та керівників, які хочуть впровадити креативні та сучасні підходи ведення справи у своїх командах та компаніях;
    • Для тих, хто бачить інновації як вирішальні для розвитку бізнесу чи власної справи.

    Модуль 1: Креативне мислення для підприємців (онлайн)
    • Управління креативними ідеями та створенням рішень
    • Як наш мозок генерує ідеї
    • Формування креативності як навички
    • Основи нейрокреативності
    • Простір для креативних ідей: проблема, контекст, рішення
    Модуль 2: Пошук та тестування бізнес-моделі (офлайн)
    • Ідентифікація та усвідомлення потреб потенційних клієнтів
    • Методи опитування клієнтів для аналізу ринку
    • Дослідження бізнес-моделей
    • Перевірка бізнес-моделі
    Модуль 3: Бізнес-стратегія (офлайн)
    • Стратегія бізнесу, що це і як її будувати?
    • Постановка цілей та вибір ринку
    • Як виграти на ринку
    • Від стратегії до чіткого плану та успішної реалізації
    Модуль 4: Маркетинг, що драйвить бізнес (онлайн)
    • Маркетинг стратегія в дії
    • Побудова сильного бренду
    • Ефективна комунікація. Робота з підрядниками
    • Інструменти просування: діджитал маркетинг та PR
    Модуль 5: Утримання та побудова відносин з клієнтами (офлайн)
    • Чому важливий рівень задоволеності клієнтів
    • Стратегії побудови міцних відносин з клієнтами
    • Впровадження механізмів зворотного зв’язку
    • Досконале розв'язання проблем клієнтів
    Модуль 6: Команда. Мотивація. Лідерство (онлайн)
    • Стадії розвитку команд
    • Теорії мотивації. Внутрішня та зовнішня мотивація
    • Ефективна комунікація та зворотний зв'язок. Feedback vs foreback
    • Стилі лідерства
    • Управління конфліктами в команді. Спроможність команд
    Модуль 7: Фінансовий процес. FinModel (онлайн)
    • Основи фінансів для підприємців
    • Інтерпретація фінансової звітності (на основі реальних кейсів)
    • Планування фінансового майбутнього бізнесу
    • Фінансовий аналіз та прийняття рішень
    Модуль 8: Пітчинг та нетворкінг (офлайн)
    • Удосконалення техніки пітчингу для залучення інвесторів і партнерів
    • Підготовка пітчдеку
    • Пітчинг
    • Нетворкінг та закриття програми. Вручення сертифікатів
